uploadUserProfile
用户信息上报接口，调用时机由开发者自定义

请求示例

复制var ui = {
  ak: 'testappkey',
  sdt: 'h5mp',
  aid: '',
  uin: 'testusernick',
  uia: 'testuseravatar',
  uig: 'female',
  uit: 'China',
  uip: 'Beijing',
  uic: 'Beijing',
  uil: 'Chinese',
  id: 'atestUserid',
  it: 'cnaid' //请与aplus-idtype值保持一致
};
aplus.uploadUserProfile(JSON.stringify(ui), function(res) {
  console.log('uploadUserProfile: ', res);
});
请求参数：
 
参数          类型        必填               描述                         默认值
ui           string       是       用户信息Object的json string        0
ak           string       是             appkey
sdt          string       是             sdktype
aid          string       否              appid
uin          string       否              用户昵称
uia          string       否              用户头像
uig          string       否              用户性别
uit         string        否              用户国家
uip         string        否              用户省份
uic         string        否              用户城市
uil         string        否              用户语言
id          string       否              用户id
it         string        否              用户id类型
